# PostgreSQL Schema Conflict Detector

## Описание

Прототип системы семантического обнаружения конфликтов при трёхстороннем объединении параллельных преобразований схем реляционных баз данных PostgreSQL.

Система:

- строит графовую модель схемы базы данных;
- реконструирует операции изменения схемы;
- обнаруживает семантические конфликты;
- анализирует возможность трёхстороннего объединения;
- проверяет инварианты целостности;
- формирует текстовые и JSON-отчёты.

## Основные возможности

- реконструкция изменений схемы;
- обнаружение конфликтов переименования;
- обнаружение конфликтов удаления и модификации;
- обнаружение конфликтов ссылочной целостности;
- анализ зависимостей объектов схемы;
- проверка инвариантов после объединения;
- анализ коммутативности операций;
- генерация отчётов.
